# 金融分析工具库 — 学习版总览

这是 Anthropic 官方的 Claude 金融服务业参考实现，包含 **66 个分析技能**，覆盖 7 个金融垂直领域。

## 学习路线建议

### 第一阶段：核心建模（必读）
先学 01-金融分析，这是所有其他技能的基础：

1. **01-comps-可比公司分析** — 估值的起点，所有交易都要用
2. **02-dcf-估值模型** — 企业估值的核心方法
3. **03-lbo-杠杆收购** — PE视角的估值
4. **04-三表模型** — 财务预测的基础
5. **05-Excel审计** — 保证模型质量

### 第二阶段：按你的方向深入

| 方向 | 推荐路径 |
|------|---------|
| **卖方研究** | 02-权益研究（财报分析→覆盖启动→投资论点跟踪） |
| **投行** | 03-投行（CIM→Teaser→买家清单→并购模型） |
| **买方PE** | 04-私募股权（项目筛选→尽调→IC备忘录→投后监控） |
| **财富管理** | 05-财富管理（财务规划→组合再平衡→税损收割） |
| **基金运营** | 06-基金管理（总账对账→断点追溯→NAV核对） |

### 第三阶段：定制你自己的技能
学完后用 01-金融分析/13-技能创建器 的框架，把你的分析方法论写成 Claude 可执行的技能。

## 每个技能文件夹的内容

- **README.md** — 学习指南（中文，说明方法论和学习要点）
- **SKILL.md** — 核心方法论（英文，这是你要学的重点）
- **command.md** — 命令定义（如何调用这个技能）
- **references/** — 参考资料（如有）

## MCP 数据源

技能中引用的数据来自以下 MCP 连接器（需要订阅或API Key）：

| 数据源 | 用途 |
|--------|------|
| FactSet | 财务数据、估值、共识预期 |
| Morningstar | 基金数据、股票分析 |
| S&P Global | 公司数据、信用评级 |
| Moody's | 信用风险、固定收益 |
| PitchBook | PE/VC数据、交易 |
| LSEG | 市场数据、固收、外汇 |
| Daloopa | 自动化财务数据提取 |
| Aiera | 财报电话会议转录 |
| MT Newswires | 实时新闻 |
| Chronograph | PE投后数据 |
| Egnyte | 文档管理 |

## 关键概念

- **Skill（技能）**= 分析方法论 + 步骤 + 质量检查 + 输出格式
- **Command（命令）**= 调用技能的斜杠命令（如 /comps、/dcf）
- **MCP（Model Context Protocol）**= Claude 连接外部数据源的协议
- **Agent（Agent）**= 端到端工作流，组合多个技能
