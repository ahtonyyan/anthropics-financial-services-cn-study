# 财报分析 (Earnings Analysis)

## 一句话说明
季报出来后，拉财报+电话纪要+SEC文件，做Beat/Miss分析，生成8-12页研报

## 核心方法论
1. **验证时效性** — 确认是最新季度数据
2. **Beat/Miss分析** — 营收 vs 共识、EPS vs 共识、分部表现
3. **关键指标** — 收入拆分、利润率趋势、指引变化
4. **生成图表** — 8-12张：收入/EPS趋势、利润率、分部收入、估值
5. **输出研报** — 8-12页DOCX，含评级和目标价

## 学习要点
- **Beat/Miss要量化**：不是"好于预期"，是"超出3.2%"
- **指引比结果更重要**：市场看的是未来，不是过去
- **图表是核心**：8-12张图让报告有说服力
- **来源要可点击**：每个数字都要标注出处

## 关联技能
- 依赖模型更新技能调整预测
- 财报前瞻提供预期基准
- 投资论点跟踪记录变化

## 适用场景
季报发布后24-48小时内出研报
