# 权益研究 (Equity Research)

卖方/买方研究分析师的核心工作流，从财报分析到投资论点管理。

## 包含技能 (9个)

| 编号 | 技能 | 命令 | 说明 |
|------|------|------|------|
| 01 | earnings-财报分析 | /earnings | 季报分析→8-12页研报 |
| 02 | earnings-preview-财报前瞻 | /earnings-preview | 财报前预测+情景分析 |
| 03 | 覆盖启动报告 | /initiate | 首次覆盖全面研报 |
| 04 | 模型更新 | /model-update | 用新数据更新预测 |
| 05 | 晨会纪要 | /morning-note | 每日市场动态+交易思路 |
| 06 | 行业全景 | /sector | 行业深度报告 |
| 07 | 投资论点跟踪 | /thesis | 持续跟踪论点变化 |
| 08 | 催化剂日历 | /catalysts | 事件驱动日历 |
| 09 | 选股筛选 | /screen | 量化筛选+研究思路 |

## 典型工作流

```
选股筛选 → 覆盖启动报告 → 财报前瞻 → 财报分析 → 模型更新 → 投资论点跟踪
                                            ↑
晨会纪要（每日）  催化剂日历（持续）        行业全景（季度）
```

## 学习顺序

01(财报分析) → 03(覆盖启动) → 07(论点跟踪) → 其余按需
