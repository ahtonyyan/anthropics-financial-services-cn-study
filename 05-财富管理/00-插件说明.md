# 财富管理 (Wealth Management)

理财顾问的核心工作流——从客户报告到投资规划。

## 包含技能 (6个)

| 编号 | 技能 | 命令 | 说明 |
|------|------|------|------|
| 01 | 客户报告 | /client-report | 面向客户的业绩报告 |
| 02 | 客户会议准备 | /client-review | 简报包+talking points |
| 03 | 财务规划 | /financial-plan | 退休/教育/遗产规划 |
| 04 | 投资提案 | /proposal | 潜在客户方案 |
| 05 | 组合再平衡 | /rebalance | 偏离度+税务感知调仓 |
| 06 | 税损收割 | /tlh | TLH机会+wash sale |

## 典型工作流

```
财务规划 → 投资提案 → 组合再平衡 → 税损收割
                ↓
        客户报告 → 客户会议准备（定期）
```

## 学习顺序

03(财务规划) → 05(再平衡) → 06(TLH) → 其余按需
