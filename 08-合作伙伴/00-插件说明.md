# 合作伙伴插件

第三方数据提供商的技能集成。

## LSEG (8个技能)

| 编号 | 技能 | 说明 |
|------|------|------|
| 01 | 债券相对价值 | 债券定价分析 |
| 02 | 债券期货基差 | 期货-现货套利 |
| 03 | 互换曲线策略 | 利率曲线交易 |
| 04 | FX套利 | 外汇carry trade |
| 05 | 期权波动率 | IV曲面分析 |
| 06 | 宏观利率监控 | 央行政策跟踪 |
| 07 | 股票研究 | LSEG数据的股票分析 |
| 08 | 固收组合 | 债券组合审查 |

## S&P Global (3个技能)

| 编号 | 技能 | 说明 |
|------|------|------|
| 01 | tear-sheet | 公司一页快照 |
| 02 | earnings-preview | 财报前瞻 |
| 03 | funding-digest | 融资活动摘要 |

## 数据源

需要 LSEG 或 S&P Global 的 MCP 订阅/API Key。
