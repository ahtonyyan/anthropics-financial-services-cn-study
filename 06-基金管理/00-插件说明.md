# 基金管理 (Fund Admin)

基金运营团队的核心工作流——GL对账、NAV核对、月末结账。

## 包含技能 (6个)

| 编号 | 技能 | 说明 |
|------|------|------|
| 01 | 应计调度 | 收入/费用的应计确认 |
| 02 | 断点追溯 | 对账差异的根因定位 |
| 03 | 总账对账 | GL vs 子账逐笔匹配 |
| 04 | NAV核对 | 基金净值核对 |
| 05 | 滚动结转 | 期初→期末变化追踪 |
| 06 | 差异评述 | 预算vs实际差异解释 |

## 典型工作流

```
总账对账 → 断点追溯 → 差异评述 → 月末结账
    ↓
NAV核对 → LP报表分发
```

## 学习顺序

03(总账对账) → 02(断点追溯) → 05(滚动结转) → 其余按需
