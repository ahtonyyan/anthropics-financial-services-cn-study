# Agent 模板

这些是 **部署配置**，不是学习重点。每个 Agent 是一个端到端工作流，组合了多个技能。

## 10 个 Agent

| Agent | 垂直领域 | 功能 | 子Agent |
|-------|---------|------|---------|
| pitch-agent | 投行 | Comps+LBO→品牌pitch deck | researcher, modeler, deck-writer |
| market-researcher | 权益研究 | 行业→竞争格局+同业+思路 | sector-reader, comps-spreader, note-writer |
| earnings-reviewer | 权益研究 | 财报→模型更新→研报 | transcript-reader, model-updater, note-writer |
| meeting-prep-agent | 财富管理 | 客户会议简报包 | profiler, news-reader, pack-writer |
| model-builder | 金融分析 | DCF/LBO/三表→Excel | data-puller, builder, auditor |
| gl-reconciler | 金融分析 | 总账对账→断点→签字 | reader, critic, resolver |
| kyc-screener | 金融分析 | KYC文档→规则→标记 | doc-reader, rules-engine, escalator |
| valuation-reviewer | 私募 | GP估值包→LP报告 | package-reader, valuation-runner, publisher |
| month-end-closer | 金融分析 | 应计/滚动/差异 | ledger-reader, rollforward, poster |
| statement-auditor | 私募 | LP报表审计 | statement-reader, reconciler, flagger |

## 文件结构

每个 Agent 目录下：
- `agent.yaml` — 主配置（系统提示+技能+子Agent）
- `subagents/*.yaml` — 子Agent配置
- `steering-examples.json` — 交互示例
- `README.md` — 安全级别和注意事项

## 部署方式

```bash
export ANTHROPIC_API_KEY=sk-ant-...
scripts/deploy-managed-agent.sh <agent-name>
```

## 何时关注这部分

当你学完所有技能，想把它们组合成自动化工作流时再来看这部分。
