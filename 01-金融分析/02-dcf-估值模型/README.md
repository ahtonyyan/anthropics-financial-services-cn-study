# DCF估值模型

## 一句话说明
预测未来5-10年自由现金流，用WACC折现回现值，加上终值，得出企业价值和隐含股价

## 核心方法论
1. **先跑Comps** — 用同业倍数校准终值假设（这是关键创新点）
2. **收入预测** — Bear/Base/Bull三情景
3. **利润建模** — OpEx、EBITDA、FCF
4. **算WACC** — CAPM模型，Beta、无风险利率、ERP
5. **折现+终值** — 终值占EV的50-70%是正常范围
6. **交叉验证** — 隐含EV/EBITDA vs 同业中位数

## 学习要点
- **终值是灵魂**：终值占比太高说明近期现金流不值钱，太低说明你对远期太乐观
- **WACC的敏感性**：WACC变0.5%，估值可能变20%
- **三情景不是摆设**：Bear/Base/Bull的假设要有逻辑差异，不是拍脑袋
- **Comps交叉验证**：DCF算出30倍EV/EBITDA但同业只给12倍？说明你的假设有问题

## 关联技能
- 依赖 comps-analysis（先跑comps再做DCF）
- LBO模型用类似方法但视角不同
- 3-statement-model提供详细的财务预测

## 适用场景
公司估值、投资决策、目标价设定
