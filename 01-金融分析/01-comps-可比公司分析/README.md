# 可比公司分析 (Comps Analysis)

## 一句话说明
选4-6家同业公司，拉取运营指标和估值倍数，用统计分布（中位数/四分位）判断目标公司是高估还是低估

## 核心方法论
1. **选同业** — 业务模式、规模、行业、地域四维度匹配
2. **拉数据** — 营收、增长、毛利率、EBITDA、市值、企业价值
3. **算倍数** — EV/Revenue、EV/EBITDA、P/E
4. **做统计** — Max、75th、Median、25th、Min
5. **输出Excel** — 蓝色=输入、黑色=公式、无hardcode

## 学习要点
- **同业选择逻辑**：不是随便挑几家，要业务可比、规模可比
- **倍数的含义**：EV/EBITDA反映的是"企业整体估值 vs 盈利能力"
- **统计分布的意义**：中位数比均值更抗异常值，四分位告诉你正常范围
- **行业特有指标**：SaaS看Rule of 40，零售看同店销售，金融看ROE

## 关联技能
- DCF模型的终值假设依赖 comps 的中位数倍数
- LBO的退出倍数也参考 comps
- 估值复盘用 comps 做交叉验证

## 适用场景
并购估值、IPO定价、投资分析、行业对标
