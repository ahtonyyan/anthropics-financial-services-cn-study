# Excel模型审计

## 一句话说明
检查Excel财务模型的公式错误、hardcode、平衡问题——投行分析师的基本功

## 核心方法论
1. **公式追踪** — 每个公式是否引用正确
2. **Hardcode检测** — 公式里藏了直接输入的数字
3. **平衡检查** — 资产=负债+权益
4. **一致性** — 单位、时间轴、假设是否统一
5. **敏感性** — 关键假设变化时模型反应是否合理

## 学习要点
- **Hardcode是大忌**：公式里的数字必须标记来源
- **审计清单**：系统化检查，不要靠记忆
- **蓝色=输入，黑色=公式**：行业规范，一眼看出哪些是假设

## 关联技能
- 三表模型、DCF、LBO都需要审计
- 数据清洗是审计的前置步骤

## 适用场景
模型交付前质检、接手他人模型时检查
