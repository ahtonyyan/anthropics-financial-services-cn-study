# 三表联动模型

## 一句话说明
利润表、资产负债表、现金流量表三表联动，一个数字变了其他两张表自动跟着变

## 核心方法论
1. **历史数据** — 3-5年历史财报
2. **驱动假设** — 收入增长、利润率、周转率、资本支出
3. **利润表预测** — 收入→成本→EBITDA→净利润
4. **资产负债表** — 资产=负债+权益，平衡检查
5. **现金流量表** — 从净利润倒推经营/投资/筹资现金流
6. **循环引用** — 现金→利息→净利润→留存收益→现金

## 学习要点
- **三表必须平衡**：资产=负债+权益，不平衡就是错了
- **循环引用处理**：现金余额影响利息，利息影响净利润，净利润影响现金
- **驱动假设要合理**：不是拍脑袋，要参考历史和同业
- **Excel建模规范**：蓝色输入、黑色公式、无hardcode

## 关联技能
- DCF和LBO都依赖三表模型的预测
- Excel审计技能检查三表模型质量

## 适用场景
财务预测、估值基础、情景分析
