# LBO杠杆收购模型

## 一句话说明
模拟用大量债务收购一家公司，看退出时能赚多少IRR/MOIC——PE基金的核心分析工具

## 核心方法论
1. **假设收购价** — 通常在当前EV基础上加溢价
2. **设计资本结构** — 优先级债、次级债、夹层、股权的比例
3. **预测现金流** — 关键是公司能不能还得起利息和本金
4. **假设退出** — 5-7年后按什么倍数退出
5. **算回报** — IRR（内部收益率）和MOIC（投资倍数）

## 学习要点
- **杠杆是双刃剑**：债务越多，股权回报越高，但风险也越大
- **现金流是生命线**：LBO模型的核心不是估值，是"还得起钱吗"
- **退出倍数假设**：通常参考当前comps，假设不变或略低
- **IRR vs MOIC**：IRR考虑时间价值，MOIC只看倍数，两者都要看

## 关联技能
- Comps提供退出倍数参考
- 3-statement提供详细预测
- IC-memo需要LBO结果作为回报分析

## 适用场景
PE收购评估、交易结构设计、回报分析
