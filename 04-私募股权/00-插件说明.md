# 私募股权 (Private Equity)

从项目发掘到投后管理的PE全流程。

## 包含技能 (10个)

| 编号 | 技能 | 命令 | 说明 |
|------|------|------|------|
| 01 | 项目发掘 | /source | 主动搜索投资标的 |
| 02 | 项目筛选 | /screen-deal | 快速Pass/Fail |
| 03 | 尽调清单 | /dd-checklist | 按工作流生成清单 |
| 04 | 尽调会议准备 | /dd-prep | 管理层演示准备 |
| 05 | 单元经济 | /unit-economics | SaaS核心指标 |
| 06 | 回报分析 | /returns | IRR/MOIC敏感性 |
| 07 | IC备忘录 | /ic-memo | 投委会审批材料 |
| 08 | 投后监控 | /portfolio | KPI跟踪 |
| 09 | 价值创造计划 | /value-creation | 100天计划 |
| 10 | AI就绪度 | /ai-readiness | AI应用潜力评估 |

## 典型工作流

```
项目发掘 → 项目筛选 → 尽调清单 → 尽调会议准备 → 单元经济 → 回报分析 → IC备忘录
                                                                      ↓
                                                              投后监控 → 价值创造计划
```

## 学习顺序

02(筛选) → 07(IC备忘录) → 06(回报分析) → 05(单元经济) → 其余按需
