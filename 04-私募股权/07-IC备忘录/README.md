# IC备忘录 (Investment Committee Memo)

## 一句话说明
投委会审批材料——综合尽调发现、财务分析、交易条款，给出投资建议

## 核心方法论
1. **执行摘要** — 一页纸说清推荐理由和回报
2. **公司概览** — 业务、行业、竞争
3. **财务分析** — 历史+预测+估值
4. **尽调发现** — 商业/财务/法律/运营
5. **交易条款** — 价格、结构、保护条款
6. **价值创造** — 100天计划、EBITDA桥
7. **风险和缓释** — Top 3风险+应对方案

## 学习要点
- IC memo要"结论先行"，推荐/不推荐在第一行
- 风险部分要诚实，不要粉饰
- 回报分析要有多情景

## 关联技能
- 依赖尽调清单、回报分析、单元经济等多个技能
