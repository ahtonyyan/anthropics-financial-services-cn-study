# 投行 (Investment Banking)

并购交易全流程的材料和工具——从CIM到pitch deck。

## 包含技能 (9个)

| 编号 | 技能 | 命令 | 说明 |
|------|------|------|------|
| 01 | CIM起草 | /cim | 保密信息备忘录 |
| 02 | teaser-匿名简介 | /teaser | 一页匿名公司介绍 |
| 03 | 买家清单 | /buyer-list | 战略+财务买家列表 |
| 04 | 并购模型 | /merger-model | 增厚/摊薄分析 |
| 05 | 流程函 | /process-letter | 竞标指引 |
| 06 | 公司一页 | /one-pager | Strip profile |
| 07 | pitch-deck | — | Pitch book生成 |
| 08 | 数据包 | — | 从CIM提取结构化数据 |
| 09 | 交易跟踪 | /deal-tracker | 里程碑+待办管理 |

## 典型工作流

```
公司一页 → Pitch Deck → CIM → Teaser → 买家清单 → 流程函 → 并购模型
                                                        ↓
                                                  交易跟踪（全程）
```

## 学习顺序

01(CIM) → 06(公司一页) → 02(Teaser) → 04(并购模型) → 其余按需
