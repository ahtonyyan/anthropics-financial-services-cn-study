# CIM保密信息备忘录

## 一句话说明
卖方最重要的文件——匿名描述公司业务、财务、市场，供潜在买家评估

## 核心方法论
1. **执行摘要** — 最吸引人的一页
2. **业务概述** — 产品、客户、竞争地位
3. **行业背景** — 市场规模、增长、趋势
4. **财务分析** — 3-5年历史 + 预测
5. **投资亮点** — 为什么值得买

## 学习要点
- CIM是"卖故事"，不是"列数据"
- 匿名处理很重要，不能泄露公司身份
- 财务部分要扎实但乐观

## 关联技能
- 数据包从CIM中提取结构化数据
- Teaser是CIM的匿名摘要版

## 适用场景
并购出售流程、私募融资
