# FINSKILLS 学习站设计文档

## 概述
一个纯静态 HTML+JS 网站，为 62 个金融 Skills 提供详细的中文学习页面，用人话讲解 + 案例展示。

## 技术栈
- 纯静态：单个 HTML 文件 + 内联 CSS/JS
- 无依赖，双击即可打开
- 数据从各 SKILL.md 文件中提取，内嵌为 JSON

## 覆盖范围
8 个领域共 62 个 Skills：
- 01-金融分析 (13) / 02-权益研究 (9) / 03-投行 (9) / 04-私募股权 (10)
- 05-财富管理 (6) / 06-基金管理 (6) / 07-运营KYC (2) / 08-合作伙伴 (11)

## 视觉设计
- **风格**：浅色编辑风格，温暖米白底色
- **配色**：暖白 `#f7f5f0` 背景，金色 `#c8a44e` 品牌色，深色顶部导航 `#1a1a2e`
- **字体**：Georgia/Noto Serif SC 正文，系统字体 UI 元素
- **字号**：正文 16-17px，板块标题 20px，页面标题 36px
- **布局**：左侧领域导航 (240px) + 右侧详情内容区

## 页面结构

### 全局导航
- 顶部深色导航栏：品牌名 FINSKILLS | 技能库 | 搜索
- 左侧边栏：按领域分组，显示领域名 + skill 数量，点击展开 skill 列表
- 当前选中项用金色左边框 + 白色背景高亮

### Skill 详情页 — 5 个板块
1. **白话讲解** (💡 金色) — 用通俗语言解释 skill 是什么、解决什么问题
2. **什么时候用** (🎯 绿色) — 触发场景标签列表
3. **怎么用** (📋 黄色) — 带编号的步骤流程
4. **案例展示** (🔍 红色) — 输入→处理→输出的可视化案例
5. **关键概念** (📖 紫色) — 可点击的专业术语，点击弹出解释

### 交互行为
- 左侧导航点击切换 skill，右侧内容动态更新（不刷新页面）
- 搜索框实时过滤 skills
- 关键概念点击弹出浮层解释
- 面包屑导航显示当前位置

